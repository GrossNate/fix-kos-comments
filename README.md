# fix-kos-comments

Daily Kos beta threaded-comments bookmarklet.

## Files
- `dailykos-threaded-bookmarklet.js`: readable source.
- `dist/dailykos-threaded-bookmarklet.txt`: one-line bookmarklet payload.

## Install
1. Open your browser bookmarks manager.
2. Create a new bookmark.
3. Set its URL to the full contents of `dist/dailykos-threaded-bookmarklet.txt`.

## Use
1. Go to a Daily Kos beta story page (for example: `https://beta.dailykos.com/stories/...`).
2. Click the bookmarklet.
3. It replaces `#dk-comments` with a threaded renderer.
4. Click the bookmarklet again (or use `Restore Viafoura`) to revert.
